# Ren Echo 工具箱

免费在线开发者工具箱，纯前端静态页面，所有数据在浏览器本地处理。

## 包含工具（15个）

| 工具 | 说明 |
|------|------|
| JSON 格式化 | 美化、压缩、验证 JSON |
| Base64 编解码 | 文本 Base64 编码/解码 |
| 颜色转换器 | HEX / RGB / HSL 互转 |
| 时间戳转换 | Unix 时间戳与日期互转 |
| UUID 生成器 | 批量生成 UUID v4 |
| Hash 计算器 | SHA-1/SHA-256/SHA-384/SHA-512 |
| 正则表达式测试 | 实时匹配高亮、捕获组 |
| URL 编解码 | URL 编码/解码 |
| Markdown 预览 | 实时渲染预览 |
| 文本对比 | 逐行差异对比 |
| 进制转换 | 二/八/十/十六进制 + 位视图 |
| 密码生成器 | 安全随机密码 + 强度检测 |
| Cron 解析器 | 表达式可视化 + 预设 |
| Lorem 生成器 | 占位文本生成 |
| IP 查询 | 本机 IP + 归属地查询 |

## 特性

- ✅ 纯前端，无需后端服务器
- ✅ 数据不离开浏览器
- ✅ 深色/浅色主题切换
- ✅ 响应式设计，支持移动端
- ✅ 搜索和分类筛选
- ✅ 一键复制结果

## 部署

直接推送到 GitHub，通过 GitHub Actions 自动部署到 GitHub Pages。

## 技术栈

- Vue 3 (CDN)
- Marked.js (Markdown 渲染)
- 纯 CSS (无 Tailwind 构建)
- GitHub Pages + GitHub Actions
